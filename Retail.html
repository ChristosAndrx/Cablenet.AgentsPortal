For Android phone:
  1. Find the application "SIM Toolkit" on your device
  2. Select Multi IMSI
  3. Select Manual Switch
  4. Select Roaming
  5. Restart your device
  6. If it still doesn't connect to a Mobile Network, go to Mobile Settings --> Mobile Network --> Network Operators and select manually an operator
For iOS Device:
  1. Go to Settings
  2. Find Mobile Data
  3. SIM Applications
  4. Select Multi IMSI
  5. Multi SIM
  6. Roaming
  7. If it still doesn't connect to a Mobile Network, go to Mobile Settings --> Mobile Network --> Network Operators and select manually an operator.
