Title: Modem Arris [Parental Control Guide]

Dear Valued Customer,

Here are the instructions on how to block adult content on our devices/modem while using alternative DNS addresses via OpenDNS.

For the Arris modem you have to:
  1. Open a Browser (Google Chrome/Firefox/Safari etc)
  2. On the address bar type: 192.168.0.1
    • User Name: admin (Lowercase Letters)
    • Password: password (Lowercase Letters)
  3. Click on Apply
  4. Click on Lan Setup
  5. Select Enable DNS Override and add:
    • Primary DNS Server IP to: 208.67.220.123
    • Secondary DNS Server IP to: 208.67.222.123
  6. Click on Apply
The modem will restart automatically and you have parental control enabled for all connected devices

Title: Bridge Mode

Dear Valued Customer,

Here are the instructions on how to set your modem to Bridge Mode.
  1. Connect your device with ethernet cable to our modem.
  2. On the address bar type: 192.168.0.1
    • User Name: admin (Lowercase Letters)
    • Password: (empty)
  3. In Basic Setup, disable both 2,4GHz & 5GHz WiFi
  4. in Wan Setup, select Dynamic & write down:
    • IP Address
    • Subnet Mask
    • Gateway Address
  5. Then Select Static and enable Static IP (Insert what you have from Dynamic settings):
    • Primary DNS: 213.140.208.226
    • Secondary DNS: 213.140.209.226
  6. Click on Apply
  7. In Wan Setup, select Dynamic and then:
    1. Disable DHCP
    2. Select Apply
  8. in Lan Setup:
    1. Disable DHCP Server
    2. NAT --> Change "RoutedWithNAT" to "Bridged"
Wait for the changes to apply and restart your modem. It should be in Bridge mode

Best regards,
Cablenet Technical Support
